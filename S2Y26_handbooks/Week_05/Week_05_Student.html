“Designing an Optimal Wildlife Corridor”

SCENARIO

The Department of Biodiversity is designing a rectangular wildlife corridor to connect two nature reserves, with 2000 meters of fencing available.

Ecologists want a large corridor, but not a very narrow one because edge effects reduce habitat quality.

The Challenge: Design a corridor that: - Uses exactly 2000 m of fencing (perimeter constraint) - Maximizes usable area - Meets minimum width requirement of 200 m (prevents edge effects)

YOUR TASK

Part A: Setting Up the Optimization (15 minutes)

  1. Let \(x\) = width of corridor (meters) and \(y\) = length (meters).

  2. Write the constraint equation for perimeter: \[2x + 2y = 2000\]

  3. Solve the constraint for \(y\) in terms of \(x\): \[y = ?\]

  4. Write the objective function for area: \[A(x) = x \cdot y\]

    Substitute your expression for \(y\) to get \(A\) as a function of \(x\) only.

  5. What is the domain of \(x\)?

    • Mathematical: \(x > 0\) (width must be positive)
    • Physical: \(x \geq 200\) (minimum width) and \(x \leq ?\) (ensure \(y > 0\))

Part B: Finding the Optimal Dimensions (25 minutes)

  1. Take the derivative: \(A'(x) = ?\)

  2. Find the critical point by solving \(A'(x) = 0\)

  3. Verify this is a maximum:

    • Calculate \(A''(x)\)
    • Evaluate \(A''(x_{critical})\)
    • Since \(A'' < 0\), this is a maximum ✓

  4. Is the critical point within the feasible range \([200, ?]\)?

    • If YES: this is the optimal width
    • If NO: the optimum is at a boundary

  5. Compare the area at:

    • The critical point (if feasible)
    • The minimum width boundary (\(x = 200\))
    • The maximum width boundary (where \(y = 200\), by symmetry)

    Which gives the largest area?

Part C: Sensitivity Analysis (15 minutes)

  1. Budget constraints may reduce available fencing to 1800 m.
    • Repeat the optimization with perimeter = 1800 m
    • How much area is lost compared to the 2000 m scenario?
  2. If the minimum width increases to 250 m (stricter ecological requirements):
    • Does this change the optimal design for 2000 m of fencing?
    • Calculate the new optimal area

Part D: Recommendation (5 minutes)

  1. Write a brief 3-4 sentence recommendation for the Department:
    • Optimal corridor dimensions
    • Expected habitat area
    • Trade-offs between budget (fencing length) and ecological requirements (minimum width)

CONCEPT REMINDERS

Use the notes below if your group needs a quick reminder of how constrained optimisation works while solving the task.

  • Use the constraint to rewrite the objective in one variable.
  • A critical point comes from setting the derivative equal to zero.
  • The mathematical optimum must still lie inside the feasible physical range.
  • When a constraint creates boundaries, compare boundary values as well as the critical point.
  • If \(A''(x) < 0\), the curve is concave down and the critical point is a local maximum.

HINTS / SCAFFOLDING

General strategy for constrained optimization: 1. Write constraint equation 2. Solve for one variable in terms of the other 3. Substitute into objective function → single-variable function 4. Take derivative and find critical points 5. Check feasibility and boundary values

For this problem: - Constraint: \(2x + 2y = 2000\) → \(y = 1000 - x\) - Objective: \(A(x) = x(1000 - x) = 1000x - x^2\) - Derivative: \(A'(x) = 1000 - 2x\) - Critical point: \(1000 - 2x = 0\) → \(x = 500\)

Python template:

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(200, 800, 500)
A = x * (1000 - x)  # Area function

# Critical point
x_crit = 500
A_crit = x_crit * (1000 - x_crit)

plt.figure(figsize=(10, 6))
plt.plot(x, A, linewidth=2, label='Area A(x)')
plt.plot(x_crit, A_crit, 'ro', markersize=10, label=f'Optimum: x={x_crit}m, A={A_crit:,}m²')
plt.axvline(x=200, color='orange', linestyle='--', alpha=0.5, label='Min width = 200m')
plt.xlabel('Width x (meters)')
plt.ylabel('Area (m²)')
plt.title('Wildlife Corridor: Area vs Width')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

“Solar Farm Economics: Optimal Panel Configuration”

SCENARIO

A renewable energy company is designing a 50-hectare solar farm near Geraldton. The farm will use rectangular arrays of solar panels. Due to shading effects and maintenance access requirements, there are engineering constraints on the configuration.

Economic model: Annual revenue (in thousands of dollars) is modeled as:

$R(x) = -2x^2 + 180x - 1600$

where $x$ is the average spacing between panel rows (in meters). This accounts for:

  • More spacing → less shading → higher efficiency per panel
  • More spacing → fewer total panels → lower total output
  • Optimal balance exists between these competing factors

Constraints:

  • Minimum spacing: $x $m (maintenance access)
  • Maximum spacing: $x $m (land area limit)
  • Panels must generate positive revenue

YOUR TASK

Part A: Critical Point Analysis (25 minutes)

  1. Find the derivative $R’(x)$ :

    • Use power rule on each term
    • Interpret: $R’(x)$ represents the rate of change of revenue with respect to spacing

  2. Find the critical point by solving $R’(x) = 0$ :

    • Set your derivative equal to zero
    • Solve for $x$
    • This gives the candidate for optimal spacing

  3. Verify this is a maximum using the second derivative test:

    • Calculate $R’’(x)$
    • Evaluate $R’’(x_{critical})$
    • If $R’’ < 0$ → local maximum ✓
    • If $R’’ > 0$ → local minimum ✗

  4. Calculate the maximum revenue:

    • Substitute $x_{optimal}$ into $R(x)$
    • Express in dollars per year

Part B: Constraint Boundaries (20 minutes)

  1. The critical point must lie within the feasible region $[8, 60]$ . Check:

    • Is $8 x_{optimal} $ ?
    • If YES: proceed to step 6
    • If NO: the optimum is at a boundary

  2. Compare revenue at three points:

    • $R(x_{optimal})$ [if within constraints]
    • $R(8)$ [minimum spacing boundary]
    • $R(60)$ [maximum spacing boundary]
    • Which configuration yields highest revenue?

  3. Find the break-even points where revenue = 0:

    • Solve: $-2x^2 + 180x - 1600 = 0$
    • Use quadratic formula: $x = $
    • Interpret: below the lower break-even point, the farm loses money

Part C: Sensitivity Analysis (15 minutes)

  1. Climate change projections suggest reduced solar irradiance will decrease revenue by 15%.

    • New model: $R_{climate}(x) = 0.85 R(x) = -1.7x^2 + 153x - 1360$
    • Find the new optimal spacing
    • How much does optimal revenue decrease?

  2. To compensate for revenue loss, the company can install higher-efficiency panels (20% boost):

    • New model: $R_{upgrade}(x) = 1.20 R(x) = -2.4x^2 + 216x - 1920$
    • Find the new optimal spacing and revenue
    • Compare to the baseline scenario

Part D: Economic Interpretation (5 minutes)

  1. Write a brief (4-5 sentence) recommendation to the company’s CEO covering:
    • Optimal panel spacing
    • Expected annual revenue
    • Sensitivity to climate change
    • Whether high-efficiency panel upgrade is justified

PYTHON TEMPLATE

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(5, 65, 500)
R = -2*x**2 + 180*x - 1600

# Derivative
dR_dx = -4*x + 180

# Critical point
x_crit = -180 / (-4)  # From dR/dx = 0
R_crit = -2*x_crit**2 + 180*x_crit - 1600

# Boundaries
R_min = -2*8**2 + 180*8 - 1600
R_max = -2*60**2 + 180*60 - 1600

plt.figure(figsize=(10, 6))
plt.plot(x, R, linewidth=2, label='Revenue R(x)')
plt.plot(x_crit, R_crit, 'ro', markersize=10, label=f'Optimal: x={x_crit:.1f}m, R=${R_crit:.0f}k')
plt.axvline(x=8, color='orange', linestyle='--', alpha=0.5, label='Min spacing (8m)')
plt.axvline(x=60, color='orange', linestyle='--', alpha=0.5, label='Max spacing (60m)')
plt.axhline(y=0, color='k', linestyle='-', linewidth=0.5)
plt.xlabel('Panel Spacing x (meters)')
plt.ylabel('Annual Revenue (thousands $)')
plt.title('Solar Farm Revenue Optimization')
plt.legend()
plt.grid(True, alpha=0.3)
plt.ylim(-500, 2500)
plt.show()

“Aquaculture Profit: Optimal Stocking Density for Barramundi”

SCENARIO

A barramundi aquaculture farm in Exmouth is optimizing its fish stocking density. The profit model accounts for:

  • Higher density → more fish → higher total yield
  • Higher density → competition for oxygen/food → higher mortality → lower individual fish weight

Profit function:

$P(d) = -0.5d^2 + 55d - 800$

where:

  • $P(d)$ = annual profit (thousands of dollars)
  • $d$ = stocking density (fish per 100 m³)

Constraints:

  • Minimum density: $d $ (economic viability)
  • Maximum density: $d $ (animal welfare regulations)

YOUR TASK

Part A: Optimization Using Calculus (25 minutes)

  1. Find the profit-maximizing stocking density:

    • Calculate $P’(d) = $
    • Solve $P’(d) = 0$ for $d$
    • This gives the critical point

  2. Confirm this is a maximum:

    • Calculate second derivative $P’’(d)$
    • Since $P’’(d) = -1 < 0$ everywhere, the function is concave down
    • Therefore the critical point is a global maximum ✓

  3. Calculate maximum profit:

    • Substitute $d_{optimal}$ into $P(d)$
    • Express in dollars per year

  4. Check feasibility:

    • Is $10 d_{optimal} $ ?
    • If NO: the constrained optimum is at a boundary

Part B: Constrained Optimization (20 minutes)

  1. Compare profit at critical point vs. boundaries:

    • $P(d_{optimal})$ [if within $[10, 80]$ ]
    • $P(10)$ [minimum density]
    • $P(80)$ [maximum density]
    • Which is the true optimum given constraints?

  2. Find the break-even density levels:

    • Solve: $-0.5d^2 + 55d - 800 = 0$
    • Multiply by -2: $d^2 - 110d + 1600 = 0$
    • Use quadratic formula
    • Interpret: farm loses money outside this range

  3. Calculate the marginal profit at current operating density $d = 40$ :

    • $P’(40) = ?$ dollars per fish (per 100 m³)
    • If $P’(40) > 0$ : increasing density increases profit
    • If $P’(40) < 0$ : decreasing density increases profit
    • What does this suggest?

Part C: Comparative Scenarios (15 minutes)

  1. A disease outbreak reduces survival rates, decreasing the profit function to:

    $P_{disease}(d) = -0.5d^2 + 55d - 1200$

    • Find the new optimal density
    • How much does profit decrease at the optimum?
    • Does the farm still break even?

  2. Installing aeration systems improves oxygen levels, changing the model to:

    $P_{aeration}(d) = -0.4d^2 + 60d - 900$

    • Find the new optimal density (note: less steep quadratic → flatter profit curve)
    • Calculate maximum profit
    • Is the aeration investment worthwhile?

Part D: Management Recommendation (5 minutes)

  1. Write a brief (4-5 sentence) advisory memo to the farm manager:
    • Recommended stocking density
    • Expected annual profit
    • Risk mitigation strategies (disease, aeration)
    • Monitoring parameters

PYTHON TEMPLATE

import numpy as np
import matplotlib.pyplot as plt

d = np.linspace(5, 90, 500)
P = -0.5*d**2 + 55*d - 800

# Derivative
dP_dd = -d + 55

# Critical point
d_crit = 55 / 1  # From dP/dd = 0
P_crit = -0.5*d_crit**2 + 55*d_crit - 800

# Disease scenario
P_disease = -0.5*d**2 + 55*d - 1200

# Aeration scenario
P_aeration = -0.4*d**2 + 60*d - 900
d_aer_crit = 60 / 0.8
P_aer_crit = -0.4*d_aer_crit**2 + 60*d_aer_crit - 900

plt.figure(figsize=(10, 6))
plt.plot(d, P, linewidth=2, label='Baseline P(d)', color='blue')
plt.plot(d, P_disease, linewidth=2, linestyle='--', label='Disease scenario', color='red')
plt.plot(d, P_aeration, linewidth=2, linestyle=':', label='Aeration upgrade', color='green')

plt.plot(d_crit, P_crit, 'bo', markersize=10, label=f'Baseline opt: d={d_crit:.0f}, P=${P_crit:.0f}k')
plt.plot(d_aer_crit, P_aer_crit, 'go', markersize=10, label=f'Aeration opt: d={d_aer_crit:.0f}')

plt.axvline(x=10, color='orange', linestyle='--', alpha=0.5)
plt.axvline(x=80, color='orange', linestyle='--', alpha=0.5)
plt.axhline(y=0, color='k', linestyle='-', linewidth=0.5)

plt.xlabel('Stocking Density d (fish per 100 m³)')
plt.ylabel('Annual Profit (thousands $)')
plt.title('Barramundi Aquaculture: Stocking Density Optimization')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
