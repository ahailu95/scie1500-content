## Key Formulas Summary

| Concept | Formula |
|---------|---------|
| Market equilibrium | Set $Q_d = Q_s$ |
| Max willingness to pay | Set $Q_d = 0$, solve for $P$ |
| Min acceptable price | Set $Q_s = 0$, solve for $P$ |
| Consumer Surplus (linear) | $CS = \frac{1}{2} Q^* (P_{\max} - P^*)$ |
| Producer Surplus (linear) | $PS = \frac{1}{2} Q^* (P^* - P_{\min})$ |
| CS (integral form) | $CS = \int_0^{Q^*} D(Q)\,dQ - P^* \cdot Q^*$ |
| PS (integral form) | $PS = P^* \cdot Q^* - \int_0^{Q^*} S(Q)\,dQ$ |
| LP objective | $Z = c_1 x + c_2 y$ |
| Corner Point Theorem | Optimal at vertex of feasible region |
